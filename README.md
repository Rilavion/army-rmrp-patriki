# ВС РФ · в/ч №12132 — сайт воинской части

Статический сайт для 1-й ОБрСпН (в/ч №12132). Хостинг — GitHub Pages. Опциональная авторизация и БД через Supabase.

## Структура

```
/
├── index.html          — главная (hero, о части, статистика, уставы, новости)
├── ustav.html          — уставы (ваш готовый файл, положите сюда)
├── gallery.html        — галерея с фильтрами и lightbox
├── news.html           — лента новостей + админ-панель для авторизованных

├── css/style.css       — общие стили, темы, компоненты, адаптив
├── js/
│   ├── main.js         — фоновая анимация (canvas), тема, бургер, reveal, переходы
│   ├── auth.js         — Supabase-авторизация (опционально)
│   ├── news.js         — загрузка новостей (Supabase → fallback на демо-данные)
│   └── config.js       — ключи Supabase (по умолчанию пусто)
├── data/               — HTML-фрагменты уставов для ленивой подгрузки

└── README.md
```

## Быстрый старт

1. Скопируйте все файлы в репозиторий на GitHub.
2. Settings → Pages → Source: `main` branch / root.
3. Откройте `https://<username>.github.io/<repo>/`.

Готово. Сайт работает без каких-либо ключей.

## Ustav.html

У вас уже есть готовый `ustav.html` — просто положите его в корень проекта. Ссылки из шапки на него уже настроены. Если хотите вынести тексты уставов в отдельные файлы для ленивой загрузки:

1. Разбейте текст каждого устава на отдельный HTML-фрагмент.
2. Положите в `data/ustav-vnutrenney.html`, `data/stroevoy.html` и т.д.
3. В `ustav.html` замените статичные `html:` в `DATA[]` на:

```js
async function loadUstavHtml(slug){
  const r=await fetch(`data/${slug}.html`);
  return r.text();
}
```

И вызывайте в `showDoc()`.

## Подключение Supabase

См. [SUPABASE.md](SUPABASE.md) — там всё пошагово: создание проекта, ключи, SQL для таблиц, RLS, создание админа.

Кратко: заполните `js/config.js`, и админка сразу заработает.

## Темизация

CSS-переменные объявлены в `:root` (тёмная) и `[data-theme="light"]` (светлая). Хотите поменять оттенок — правьте `--gold`, `--green1..3`, `--bg`, `--paper`. Всё остальное подхватится автоматически.

## Добавить новую страницу

1. Скопируйте `autopark.html` как шаблон.
2. Поменяйте `<title>`, содержимое `<main>` и путь `data-page` в навигации.
3. Добавьте пункт в `<nav>` во всех остальных файлах.

## Производительность

- Изображения — с `loading="lazy"`
- Canvas-анимация паузится при скрытии вкладки (`visibilitychange`)
- Никаких сборщиков и билдов — чистая статика
- Единственная внешняя зависимость — Supabase JS (грузится с CDN, но не блокирует работу)
