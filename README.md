# ВС РФ — в/ч №12132

Официальный сайт 1-й отдельной гвардейской бригады специального назначения (ОБрСпН), в/ч №12132.

## 🎖 О проекте

Сайт для военной ролевой организации (GTA 5 / FiveM / PMRP / Патрики) с информацией об уставах, подразделениях, технике и мероприятиях.

## 📁 Структура

```
├── index.html          # Главная HTML-страница (SPA)
├── styles.css          # Стили сайта
├── app.js              # JavaScript логика приложения
├── SUPABASE_SETUP.md   # Инструкция по подключению Supabase
└── README.md           # Этот файл
```

## 🚀 Функционал

### Готовые страницы
- **Главная** — информация о бригаде, статистика, быстрые ссылки
- **Уставы** — каталог уставов с подробным просмотром каждого документа
- **Галерея** — фотографии территории, техники и мероприятий
- **Новости** — лента новостей бригады
- **Автопарк** — каталог военной техники с статусами
- **Карта** — интерактивная карта расположения объектов

### Система авторизации
- Регистрация и вход пользователей
- Админ-панель для публикации новостей
- Интеграция с Supabase (Backend-as-a-Service)

### Будущие возможности
- Редактирование уставов через админку
- Галерея с загрузкой изображений
- Система отчётов военнослужащих
- Интерактивная карта с зонами
- Календарь мероприятий
- Форум/доска объявлений

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Supabase (PostgreSQL, Authentication, Realtime)
- **Хостинг**: GitHub Pages
- **Шрифты**: Google Fonts (Cormorant Garamond, Inter)

## 📦 Установка и запуск

### Локальная разработка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/yourusername/vsrf-site.git
cd vsrf-site
```

2. Откройте `index.html` в браузере или используйте локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve
```

3. Откройте `http://localhost:8000`

### GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта
3. Перейдите в **Settings** → **Pages**
4. Source: **Deploy from a branch** → **main** → **/ (root)**

## ⚙️ Настройка Supabase

Подробная инструкция: [SUPABASE_SETUP.md](SUPABASE_SETUP.md)

Краткий план:
1. Создайте проект на [supabase.com](https://supabase.com)
2. Получите URL и anon key из **Settings** → **API**
3. Создайте таблицы: `profiles`, `news`, `documents`, `vehicles`
4. Обновите `app.js` с реальными ключами

### Цветовая схема (CSS переменные)

```css
:root {
    --bg: #080f0b;           /* Основной фон */
    --gold: #cda85a;        /* Золотой акцент */
    --gold-b: #f0d89b;      /* Светлое золото */

    --text: #f3ecd9;        /* Основной текст */
    --muted: #b4ad96;       /* Приглушённый текст */

```

### Номер части

Все вхождения `12132` замените на ваш номер части.

### Данные

- `DOCS_DATA` — массив уставов
- `NEWS_DATA` — массив новостей  
- `VEHICLES_DATA` — массив техники
- `LOCATIONS_DATA` — массив локаций карты
